# Getting Started

## Requirements

- Python 3.9+
- Git

## Install

```bash
git clone https://github.com/wdg631/gk-aitools.git
cd gk-aitools
pip install -r tools/ai-tools/*/requirements.txt
```

## Run Tools

```bash
python tools/ai-news.py
python tools/content-generator.py
python tools/ai-tools/github-scraper/main.py --help
```
